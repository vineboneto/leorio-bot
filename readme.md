Um bot para tocar músicas no servidor no meu amigo. Usei o código do [pawel02](https://github.com/pawel02/discord-js-music-bot) como exemplo.

## Requisitos

- Node@20.11.0
- Pnpm@9.12.3

## Executar

```bash
node --env-file=.env src/main.js
```
